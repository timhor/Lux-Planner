FAQs

What benefits me from using LUX?

By using LUX, a user is able to freely plan out their journeys without the stress of having to book flights immediately. It allows the user time to plan what exactly they intend to do before making any committments to bookings.

Do I need to pay any fees to access the website?

No, LUX is a free-to-use website for every user.

How do I use this website?

For help on how to use the LUX website go to the 'Help' page located in the footer at the bottom of every webpage. If your query is unanswered there feel free to contact one of our many administrators (contact information found on the 'Contact Us' page).


Help

Signing Up and Logging In

A user can sign up to the website by creating an account, this will enable them to access additional features within the website, such as saving created journeys. In order to create an account a user must do the following:

  1. Click 'Sign Up' at top of any page
  2. Enter in a unique username, password of at least 8 characters, email, first and last name, and gender
  3. Click 'Create Account'. Upon doing so, the user will be automatically logged in

If a user already has an account, they can log into it by doing the following:

  1. Click 'Log in' at the top of any page
  2. Enter in their username and password

Note: If the entered username or password is incorrect, the user will be prompted to re-enter their details.

Creating a Journey

Creating a journey is the primary function of this website. A user can either create a trial journey if they have not yet created an account (instructions to do so are provided above) or create a journey to be saved under their account. To create a journey a user must do the following:

  1. Click either '+ New Journey' at the top of any page (except the home page) or enter in a first location in the search bar on the home page and click 'New Journey'
  2. Enter in a name for the journey, starting location, next location, and their respective arrival and departure dates
  3. If user wants to add additional stops: click 'ADD STOP' and repeat step (2) for new stop
  4. If user is signed in: click 'SAVE JOURNEY'. Upon doing so, the journey will be stored under their account
  5. If user is not signed in: click 'SIGN UP' or 'LOG IN' in order to do step (4), otherwise leave page

Note: If a user leaves page without saving journey, their created journey will not be saved and they will have to re-create their journey at a later time.
